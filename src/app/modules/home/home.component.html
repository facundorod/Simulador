Vital Signs Simulator Monitor

An educational app to understand the Vital Signs Monitoring

Developed to simulate clinical scenarios for teaching veterinary anesthesiology techniques. It helps to understand and analyze the changes of the vital signs through differents scenarios happened during anesthesic practices.

How it works?

The functionalities of SIMVet are divided into four main features:
    1. Creation, update and remove animal patients.
    2. Creation, update and remove clinical scenarios.
    3. Parameters configuration and execution of clinical scenarios through the time.
    4. Parameter visualization of clinical scenarios.

To start the simulation you should register here and start a simulation. You can choose either start with previous simulation or create a custom simulation from the begining. If you choose the custom simulation, you can configure and set up the Animal Patient, Physiological Parameters, Clinical Scenarios and every features involves in the simulation.
