Vital Signs Simulator Monitor

An educational app to understand the Vital Signs Monitoring

Developed to simulate clinical scenarios for teaching veterinary anesthesiology techniques. It helps to understand and analyze the changes of the vital signs through differents scenarios happened during anesthesic practices.

How it works?

The application support multi Parameters, Simulations Configurations, Scenarios, Patients and also Monitor Configuration.

To start the simulation you should register here and start a simulation. You can choose either start with previous simulation or create a custom simulation from the begining. If you choose the custom simulation, you can configure and set up the Patient, Parameters, Scenarios and every features involves in the simulation.
